    source dataclass App()

    Bases : Base

    PowerPoint application interface.

    This class provides a high-level interface to interact with Microsoft PowerPoint application. It manages the PowerPoint application instance and provides access to presentations and other PowerPoint features.

    Attributes

    • api : DispatchBaseClass — The underlying PowerPoint COM object.

    • app : App — Reference to self for consistency with PowerPoint's object model.

    • presentations : Presentations — Get the collection of presentations.

    Methods

    • quit — Quit the PowerPoint application.

    • unselect — Unselect any selected objects in the active window.

    source property App.presentations: Presentations

    Get the collection of presentations.

    Returns

    • Presentations — A collection of all open presentations.
